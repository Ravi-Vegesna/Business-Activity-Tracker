One-time browser setup — auto-save files without "Save As" dialog
Chrome / Edge: Go to Settings → Downloads → turn OFF "Ask where to save each file before downloading". Files will then save directly to your Downloads folder automatically.
Firefox: Go to Settings → General → Downloads → select "Save files to" and pick your folder. Uncheck "Always ask you where to save files".
After this one-time change, every export will save instantly with its timestamp filename — no dialog needed.
Save data to disk (JSON)
Saves everything — activities, goals, logs, and all goal links — as a timestamped business-tracker_YYYY-MM-DD_HH-MM-SS.json file. All three formats (JSON, Excel, PDF) use the same naming pattern and save to the same Downloads folder — disable "Ask where to save" in browser settings once (see tip above) and all exports land there automatically. Upload the JSON to Claude or Copilot for AI analysis.
Export to Excel (.xlsx)
Creates a timestamped business-tracker_YYYY-MM-DD_HH-MM-SS.xlsx with four color-coded sheets: Activities, Goals, Log, and Goal Relationships. Save it in the same folder as this HTML file. Open in Excel and use Copilot to analyse.
Export to PDF
Opens a color-coded report in a new browser tab and automatically triggers the Print dialog. In the dialog, set Destination to Save as PDF and click Save — it goes directly to your Downloads folder with no extra steps. Includes a cover page, goal cards with progress bars, colored status/priority pills, and linked log entries.
